# Sistem Monitoring Lab 17 (SML-17)

Sistem pemantauan komputer lab sekolah yang mendukung aktivitas belajar mengajar di Lab komputer.

## Struktur Folder
- `agent/`: Aplikasi client (Python) yang berjalan di PC lab.
- `server/`: Backend server (FastAPI) untuk menerima metrik.
- `dashboard/`: Web interface (React + Vite) untuk memantau semua PC.
- `md-files/`: Dokumentasi proyek (PRD, Architecture, Tasks).

## Dokumentasi Lengkap
Silakan merujuk ke folder `md-files/` untuk membaca Product Requirements Document (PRD), Arsitektur, dan daftar tugas (TASKS).
